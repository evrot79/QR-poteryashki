# Приложение для поиска утерянных кодов маркировки Честный знак

Веб-приложение на Python (Flask) для анализа расхождений между системами учета маркированных товаров.

## Описание

Приложение помогает найти потерянные коды маркировки путем сравнения данных из трех источников:
1. **Поступило по ЧЗ** - коды из системы Честный знак
2. **Продано + реализация** - коды из 1С
3. **В Наличии** - отсканированные коды с товаров

## Установка и запуск

1. Установите зависимости:
```bash
pip install flask
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте в браузере: http://localhost:5000

## Функционал

### Нормализация кодов

Приложение автоматически приводит коды к единому формату:
- Из кодов 1С убираются скобки у идентификаторов применения (01) и (21)
- Из кодов наличия удаляется часть начиная с (91) - ключ проверки

### Поиск потерях

Алгоритм поиска:
1. Приводит все коды к единому формату
2. Вычисляет разницу между поступившими и проданными кодами
3. Вычитает из разницы коды, находящиеся в наличии
4. Выводит список потерянных кодов

- `POST /find` - Поиск потерянных кодов
- `POST /normalize` - Нормализация кодов для конкретного поля
